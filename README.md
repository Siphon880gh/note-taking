Installation
---
Deploy on heroku, your own web server, or localhost.

Web server:
```
nohup node ./server.js > /dev/null 2>&1 &
```

Localhost:
```
npm start
```

Usage
---
Visit / for the front page. /notes will list all notes.

### Localhost
Port will be 3001 on localhost: `localhost:3001`
